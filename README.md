## Backend API

Enviroment:
```bash
export SCANHAND_CORS="http://localhost:3000"
export SCANHAND_YOO_SHOP_ID="1234"
export SCANHAND_YOO_SECRET_KEY="secret"
export SCANHAND_YOO_REDIRECT="http://localhost"
export SCANHAND_MONGODB_URI="mongodb://127.0.0.1:27017"
export SCANHAND_MONGODB_USER="root"
export SCANHAND_MONGODB_PASS="password"
export SCANHAND_S3_URL="https://<id>.selstorage.ru"
export SCANHAND_S3_ENDPOINT="https://s3.<region>.storage.selcloud.ru"
export SCANHAND_S3_REGION="region"
export SCANHAND_S3_ACCESS_KEY="access-key"
export SCANHAND_S3_SECRET_KEY="secret-key"
```
